This "astro" site is a memorial to Judy Rosenthal (born Judith-Ann Saks) - an artist that was widely known for her paintings of Houston's growth.

The homepage is a slide show with the images specified in the file data/slides.ts.
The art page is a slide show with images specified in the file data/artSlides.ts
The menu is specified in data/menu.ts

Some pages have photo grids, which are meant to be auto-populated from directories, so that adding a new photo requires only adding it to the directory.
Some pages are CollectionCarousels, which are also auto-populated from directories, so adding a new image is just adding it to the directory.

The family section is meant to be auto-populated from a gedcom genealogy file, plus images and descriptions as well-named text files if they exist.

In dev mode, there is some editing functionality built-in.
